CS 498 Group Project

The purpose of this website is to implement machine learning algorithms in order to help people plan for vacation. We are doing this in two key ways. The first is weather prediction. If you click on the weather tab you will be able to see what our machine learning model predicts the average temperature will be in the future.

The second way we are implementing machine learning is by attempting to predict prices for AirBnb's. Our model attempts to predict the prices for AirBnb's based on user specifications. These specifications include the number of rooms, bathrooms, beds, and the city the airbnb is in. If you wish to view the results of either model you can use the navigation bar at the top of the page, or click on the appropriate image to be taken to the right page
